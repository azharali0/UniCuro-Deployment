# Data Processing Register

## Key activities

- Student onboarding
- University verification
- Wallet and payments
- Discount personalisation
- AI opportunity matching
- Academic AI study support
- Wellbeing resource routing
- Marketplace/community moderation
- University analytics
- Marketing communications
- Security monitoring

Each activity must define:
- purpose
- lawful basis
- data categories
- processors
- retention
- DPIA requirement
- region
- risk rating
