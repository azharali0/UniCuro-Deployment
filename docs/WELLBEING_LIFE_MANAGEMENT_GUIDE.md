## Admin Features

## Super Admin Features
- Wellbeing governance
- Burnout detection performance
- Urgent support routing
- False-positive monitoring

## Important Safety Notes
This engine must not replace professional medical, emergency, counselling or crisis support.
For production, urgent/high-risk wellbeing flows should route users to local emergency and university support resources immediately.
