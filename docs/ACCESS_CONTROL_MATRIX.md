# Access Control Matrix

## Student

Can access:
- student dashboard
- own wallet
- own messages
- own academic data
- own wellbeing data
- own marketplace actions
- own identity/security controls

Cannot access:
- admin queues
- super admin governance
- other users' private records

## Admin

Can access assigned operations:
- support
- verification
- marketplace moderation
- fraud review
- content/CMS
- security admin
- launch admin

Cannot access:
- unrestricted super admin controls
- investor governance sign-off
- critical configuration without approval

## Super Admin

Can access:
- platform governance
- revenue controls
- compliance
- launch board
- security governance
- privileged approval workflows

Critical actions require:
- MFA
- reason
- audit event
- approval for sensitive actions
