# Privacy Operations Runbook

## DSAR workflow

1. Receive request.
2. Verify identity.
3. Classify request type.
4. Locate relevant data.
5. Apply exemptions if legally permitted.
6. Export/correct/delete/restrict as required.
7. Record evidence.
8. Complete within SLA.

## Standard SLA

- UK/EU GDPR: usually one month, subject to legal exceptions.
- Internal target: acknowledge within 72 hours.
- Internal target: complete within 21 days where possible.

## Supported request types

- Access
- Export
- Delete
- Correct
- Restrict processing
- Withdraw consent
