# Database Schema Guide

This build adds a production Prisma schema covering:

- Users and roles
- Student profiles
- Countries and universities
- Wallet and wallet ledger
- Tasks and deadlines
- Deals
- Marketplace listings
- Opportunities
- Subscriptions
- Support tickets
- Audit logs
- API events

The schema is a production foundation, not a final exhaustive database. Future build stages should add migrations for each operational subsystem.
